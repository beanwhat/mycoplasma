# Project Goal

The goal of this project is to develop a conservative, auditable framework for
benchmarking AMR genotype-phenotype relationships across Mollicutes.

The framework is intended to organize isolate-level evidence linking
antimicrobial susceptibility measurements, such as MIC values, to genetic marker
observations extracted from the same isolate set whenever possible. Initial work
will focus on *Mycoplasma bovis* because it provides the seed evidence base for
schema design, curation rules, and benchmark planning.

Benchmarking outputs should distinguish between observed data, curated
interpretation rules, and unsupported hypotheses. Any expansion to other
Mollicutes should preserve species, drug, method, and evidence provenance so
that marker performance is evaluated only under explicitly documented
assumptions.
